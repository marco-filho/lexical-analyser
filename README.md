# Analisador Léxico
## Palavras reservadas
- `var` => `VAR`
- `const` => `CONST`
- `if` => `IF`
- `while` => `WHL`
- `for` => `FOR`
- `break` => `BRK`
- `continue` => `CONT`
## Delimitadores
- `(` => `EPAREN`
- `)` => `DPAREN`
- `[` => `ECOLC`
- `]` => `DCOLC`
- `{` => `ECHAV`
- `}` => `DCHAV`
- `,` => `VIRG`
## Identificadores
Podem conter apenas caracteres alfanuméricos (i.e. letras e números) e não podem iniciar com números. e.g. `ID(exemple)`.
## Operadores
- `=` => `ATR`
- `+` => `MAIS`
- `-` => `MENS`
- `*` => `VEZS`
- `/` => `DIV`
- `==` => `IGL`
- `!=` => `DIF`
- `>=` => `MAIG`
- `<=` => `MNIG`
- `>` => `MAR`
- `<` => `MNR`
## Constantes
Podem ser números `NUM(123)` ou strings `STR('exemple')`.
